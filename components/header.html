Professional Header Component

This is your professional data science portfolio header component. Features include:

  • Modern gradient design with professional blue color scheme
  • Animated TM logo with shimmer effect
  • Fixed position with scroll transparency
  • Responsive mobile hamburger menu
  • Smooth hover animations and active page highlighting
  • Professional typography optimized for data science roles

Implementation Instructions

To use this header component in your existing pages:

  1. Save this file as components/header.html
  2. Copy the header HTML section (lines with portfolio-header class)
  3. Include the CSS in your existing pages or create a separate CSS file
  4. Add the JavaScript for mobile menu and scroll effects
  5. Update active states for each page
